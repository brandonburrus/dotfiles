## Description

`cog` is a local MCP server that gives AI agents persistent, semantically
searchable memory. Memories are saved as markdown files on disk and indexed in
a local Pinecone vector store, with embeddings generated by a local Ollama
model. Survives across sessions — agents can store insights, patterns, and
decisions and retrieve them later by semantic similarity.

## Tools provided

- `create-memory` — persist information, strategies, patterns, and insights
- `retrieve-memory` — fetch a specific memory by name via `memory://{name}`
- `search-memory` — semantic search across all memories by natural language query

## When to use

- Persisting useful patterns, decisions, or context across OpenCode sessions
- Recalling past architectural choices or debugging strategies
- Giving agents access to a long-term knowledge base that grows over time

## Prerequisites

- [Bun](https://bun.sh) — runtime used to execute the server
- [Ollama](https://ollama.com) — local embedding model server (must be running)
- [Docker](https://docker.com) — for the local Pinecone vector store emulator

## Setup

### 1. Install dependencies

```sh
cd ~/cog && bun install
```

### 2. Pull the embedding model

```sh
ollama pull qwen3-embedding:8b
```

### 3. Start the local Pinecone emulator

The emulator must be configured with `DIMENSION=4096` to match the embedding
model output size.

```sh
docker run -d \
  --name pinecone \
  -p 5081-6000:5081-6000 \
  -e PORT=5081 \
  -e PINECONE_HOST=localhost \
  -e DIMENSION=4096 \
  pinecone:latest
```

### 4. Set the required environment variable

Add to `~/.config/opencode/opencode.env`:

```
PINECONE_API_KEY=pclocal
```

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `PINECONE_API_KEY` | *(required)* | Set to `pclocal` for the local emulator |
| `MEMORY_PATH` | `~/.config/opencode/memory` | Directory where markdown memory files are stored |
| `PINECONE_INDEX_NAME` | `cog-memory` | Pinecone index name |
| `OLLAMA_EMBEDDING_MODEL` | `qwen3-embedding:8b` | Ollama model used to generate embeddings |
| `MEMORY_SEARCH_SCORE_THRESHOLD` | `0.7` | Minimum cosine similarity score for search results (0–1) |
| `PINECONE_HOST` | `http://localhost:5081` | Pinecone host (local emulator default) |
